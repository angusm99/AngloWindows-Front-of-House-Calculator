# Contributing

## Recommended flow

1. Pull the latest `main`
2. Start the app with `start.ps1`
3. Test changes locally in the browser
4. Commit with a clear message
5. Push to GitHub

## Commit message examples

- `Build Manus-aligned landing page`
- `Wire PDF intake into quote workflow`
- `Add GitHub workflow and templates`

## Before pushing

- Confirm the app still loads at `http://127.0.0.1:8000`
- Check that pricing and quote totals still update
- Make sure `README.md` still reflects the current state
